Date time Utility

Fetch the current date and time

To fetch Current Date and time

Use http://datetimeutil.herokuapp.com/datetime

To fetch just Current time

Use http://datetimeutil.herokuapp.com/time

To fetch just Current Date

Use http://datetimeutil.herokuapp.com/date

To fetch formatted Date time

Use http://datetimeutil.herokuapp.com/formatted?format=<format>

example:
http://datetimeutil.herokuapp.com/formatted?format=dd/mm/yy

for formatting details see dateformat documentation
